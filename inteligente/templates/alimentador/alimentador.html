Sobre o Projeto

Trabalho de Conclusão de Curso

O SmartFeeder é um sistema de alimentação automática para animais de estimação desenvolvido como projeto de TCC. Combinando hardware e software, o alimentador oferece uma solução completa para o cuidado automatizado de animais.

Utilizando tecnologias de Internet das Coisas (IoT), o sistema permite que tutores programem horários de alimentação, controlem porções e monitorem o consumo alimentar de seus animais remotamente através de um aplicativo intuitivo.

Objetivo

Proporcionar praticidade e segurança alimentar para pets, garantindo refeições regulares mesmo quando os tutores não estão em casa.

Tecnologia

Sistema embarcado com conectividade WiFi, controle via aplicativo web e automação programável com interface amigável.

Funcionalidades

Tecnologia avançada para cuidar do seu pet com praticidade e eficiência

Agendamento Automático

Programe horários específicos para alimentação e o sistema libera a porção automaticamente.

Controle Remoto

Gerencie a alimentação do seu pet de qualquer lugar através do aplicativo web.

Conectividade IoT

Conexão WiFi integrada para comunicação em tempo real com o sistema.

Controle de Porções

Defina a quantidade exata de ração para manter a dieta balanceada do seu pet.
