Sobre o Projeto

Trabalho de Conclusão de Curso

O SmartFeeder é um sistema de alimentação automática para animais desenvolvido como projeto de TCC. Combinando hardware e software, o alimentador oferece uma solução completa para o cuidado automatizado de bichos.

Utilizando tecnologias de Internet das Coisas (IoT), o sistema permite que tutores programem horários de alimentação, controlem porções e monitorem o consumo alimentar de seus animais remotamente através de um aplicativo intuitivo.

Objetivo

Proporcionar praticidade e segurança alimentar para animais, garantindo refeições regulares mesmo quando os tutores não estão em casa.

Tecnologia

Sistema embarcado com conectividade WiFi, controle via aplicativo web e automação programável com interface amigável.

Funcionalidades

Tecnologia avançada para cuidar dos seus animais com praticidade e eficiência

Agendamento Automático

Defina horários programáveis específicos para alimentação e o sistema irá liberar a porção automaticamente.

Controle Remoto

Gerencie a alimentação do seu animal de qualquer lugar através do aplicativo web.

Conectividade IoT

Conexão WiFi integrada para comunicação em tempo real com o sistema.

Controle de Porções

Defina a quantidade exata de ração para manter a dieta balanceada do seu animal.
